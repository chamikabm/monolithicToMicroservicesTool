# Monolithic to Microservice Conversion Tool

**How to Build and Start Project**

cd to Project Directory

**`Build and Start Java Spring Backend`**

```bash
mvn clean install
cd backend
mvn spring-boot:run
```
Back end will be start and run on http://localhost:8080

**`Build and Start Angular Frontend`**
```bash
cd frontend\src\main\frontend
npm start
```

Front end will be start and run on http://localhost:4200
